2. **Clarified SQLite vs Access architecture concern**: A colleague shared a Claude response suggesting Access is better than SQLite for multi-user scenarios. We clarified this is **not applicable** to SIGMA-RS because:

   - Waitress serializes writes, preventing typical SQLite concurrency issues
   - This is fundamentally different from Access/SQLite as a shared network file
